# TAG

A new Flutter application for TAG – Trucking Accounting on the Go.

## About the Project

TAG is a truck delivery and logistics management system that helps drivers and teams work together efficiently.

### Key Features

- Driver and team collaboration
- Delivery and logistics management
- Real-time task coordination
- Trucking accounting and expense tracking
- Fleet operation management
- Easy communication between drivers and managers
- Organized workflow for transportation services

## Getting Started

This project is built with Flutter and serves as the foundation for the TAG mobile application.
